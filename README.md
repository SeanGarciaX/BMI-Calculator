# BMI Calculator

A simple Streamlit website to calculate Body Mass Index (BMI).

## What it does
- Users enter weight in pounds and height in inches.
- The app calculates BMI using the standard U.S. formula.
- It explains what the BMI number means and shows the health category.

## Run locally
1. Install Streamlit if needed:
   ```bash
   pip install streamlit
   ```
2. Start the app:
   ```bash
   streamlit run Access_Calculator/bmi_calculator.py
   ```
