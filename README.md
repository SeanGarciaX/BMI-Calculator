# BMI Calculator

A simple Streamlit website to calculate Body Mass Index (BMI).

## What it does
- Users enter weight in pounds and height in inches.
- The app calculates BMI using the standard U.S. formula.
- It explains what the BMI number means and shows the health category.

## Run locally
1. Install Streamlit if needed:
   ```bash
   pip install streamlit
   ```
2. Start the app:
   ```bash
   streamlit run bmi_calculator.py
   ```

## Deploy with Streamlit
- Deploy on Streamlit Cloud by pushing this folder to a GitHub repo.
- Connect the repo in Streamlit Cloud and use `app.py` as the main file.

## Notes
- BMI is a general screening tool, not a medical diagnosis.
- For personal health advice, consult a healthcare professional.
